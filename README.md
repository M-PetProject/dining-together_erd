## 파일 설명

- .mwb 파일이 erd 파일 입니다

- dining_common : 공통 영역
- dining_member : 멤버 영역(팀, 멤버)
- dining_place  : 장소 영역(댓글 포함)
- dining_main   : 회식 영역
- dining_vote   : 투표 영역
- dining_notice : 공지사항 영역(게시판)
